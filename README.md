# AI Hakeem Lukman Telegraf Bot

To install dependencies:

```bash
bun install
```

To run:

Create a new file called `.env` in the root of the project with the following content:

```shell
TELEGRAM_BOT_TOKEN=youBotToken
GROQ_API_KEY=yourApiKey
```

```bash
bun start
```
